## Healthcare Backend System (Django + DRF)

A secure healthcare backend system built using **Django**, **Django REST Framework**, and **PostgreSQL**.  
The project supports user authentication, patient and doctor management, and secure patient–doctor mapping using JWT-based authentication.

---

## Features

- JWT Authentication (Login & Register)
- User-based access control
- Patient Management (CRUD)
- Doctor Management (CRUD)
- Secure Patient–Doctor Mapping
- Ownership-based data security
- PostgreSQL database
- Environment variable support

---

## Tech Stack

- **Backend:** Django, Django REST Framework
- **Authentication:** JWT (`djangorestframework-simplejwt`)
- **Database:** PostgreSQL
- **ORM:** Django ORM
- **Tools:** Postman, pgAdmin
